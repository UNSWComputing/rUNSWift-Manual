runswift_gdb¶

GDB can be used to debug the runswift program on the NAO.

  1. Sync with debugging symbols, run nao_sync -rg mario.

    Note

    Replace “mario” with name of your robot

    Tip

    -g flag uploads debug symbols to the nao

  2. Run runswift_gdb

  3. Use gdb as normal.
